Robô de Raspagem (Selenium)

Insira o número do CNES. O Python irá abrir o navegador em background, varrer o portal e baixar Instalações, Serviços e Habilitações para a pasta local.

Verifique o terminal do Python para acompanhar o progresso do Chrome.

Consolidação do Dicionário SIGTAP

O ETL processa os arquivos de texto posicionais da pasta Dados_02_SIGTAP_Bruto e gera um banco de dados unificado em Excel, resolvendo a Teoria dos Conjuntos da Regra Geral.

Cruzamento Relacional (Tabela Dourada)

Selecione um hospital já raspado. O motor cruzará a infraestrutura dele (Módulo 1) com as exigências do governo (Módulo 2) para gerar o Perfil Blindado de Permissões.

Importação de Remessa SISAIH (ETL)

Informe o número do CNES e selecione uma remessa TXT. O sistema converterá os dados posicionais para formato Excel estruturado, pronto para auditoria.

Verifique a pasta Dados_05A_Remessas_SISAIH_Brutas para adicionar arquivos TXT.
